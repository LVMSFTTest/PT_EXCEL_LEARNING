Excel Interface

Before writing a formula, know your workspace. Excel has 8 key parts — master these and everything else becomes easy.

The 8 Key Parts

1. Workbook

The entire file (like a physical book). One workbook = one .xlsx file. Can contain many worksheets inside.

2. Worksheets

Individual pages inside the workbook. Click the + at the bottom to add more. Right-click a tab to rename it.

3. The Ribbon

The toolbar at the top with tabs: Home, Insert, Formulas, Data, Review, View. Everything you need lives here.

4. Formula Bar

Below the ribbon. Shows the actual formula inside a selected cell (not just the result). Click here to edit formulas.

5. Name Box

Left of the Formula Bar. Shows selected cell address (A1, B3...). Type any address here and press Enter to jump there.

6. Cells, Rows, Columns

Cells are individual boxes. Columns go A, B, C... (vertical). Rows go 1, 2, 3... (horizontal). Cell A1 = column A, row 1.

Cell Address System

🏙️ Think of it like a city

Columns are street names (A, B, C...). Rows are floor numbers (1, 2, 3...). Cell C5 means Street C, Floor 5. Easy to navigate once you think of it this way.

A B C D

2 | Priya | 85 | 90 | =SUM(B2:C2)
3 | Arjun | 70 | 78 | =SUM(B3:C3)

Formulas and Basic Functions

Every formula starts with = and tells Excel to calculate. These 8 functions cover 80% of what you will ever need.

Formula Anatomy

= FUNCTION_NAME ( argument1 , argument2 , ... )

=

Tells Excel this is a formula, not plain text

SUM

The function — what Excel should do

(F2:F201)

Arguments — the data to work on

SUM — Add Everything Up

=SUM(range)
🛍️ Shopping data — total of all prices
=SUM(F2:F201) ← adds every price in the dataset =SUM(F2,F5,F10) ← add specific individual cells =SUM(F2:F10,F20:F30) ← add two separate ranges

Shortcut: Click where you want the total, press Alt+= and Excel guesses the range!

SUMIF — Add Only Matching Rows

=SUMIF( range_to_check , condition , range_to_add )
🛍️ Total sales from Shoes category only
=SUMIF(D2:D201, "Shoes", F2:F201) ↑ ↑ ↑ Category col Must equal Add prices =SUMIF(B2:B201, "Female", F2:F201) ← total spent by female customers =SUMIF(E2:E201, ">3", F2:F201) ← sum prices where quantity > 3

COUNT, COUNTA, COUNTIF

=COUNT()

Counts cells with numbers only.

=COUNT(F2:F201) → how many price entries

=COUNTA()

Counts any non-empty cell.

=COUNTA(A2:A201) → count all invoice IDs

=COUNTIF()

Count cells matching a condition.

=COUNTIF(I2:I201,"Kanyon") → visits to Kanyon mall

AVERAGE, MAX, MIN, ROUND

AVERAGE

=AVERAGE(F2:F201) ← average price =AVERAGE(C2:C201) ← average age

MAX and MIN

=MAX(F2:F201) ← most expensive =MIN(F2:F201) ← cheapest purchase

ROUND and ROUNDUP

=ROUND(3.567, 2) → 3.57 =ROUNDUP(3.1, 0) → 4 =ROUNDUP(AVERAGE(F2:F201),0)
: vs , inside a formula

=SUM(A1:A10) — colon means FROM A1 TO A10 (a range)
=SUM(A1,A3,A7) — comma means these 3 specific cells only. They are very different!

Text Functions and Dates

Real data is messy — names with extra spaces, dates in wrong formats, text crammed in one column. These functions fix all of that.

LEFT, RIGHT, MID — Extract Parts of Text

=LEFT(text, n)

First n characters from left

=LEFT("I138884",1) → "I"

=RIGHT(text, n)

Last n characters from right

=RIGHT("I138884",6) → "138884"

=MID(text, start, n)

n chars starting at position

=MID("I138884",2,3) → "138"
👤 Extract first name from "Priya Sharma"
=LEFT(A2, FIND(" ",A2)-1) → "Priya" Step by step: FIND(" ",A2) → space is at position 6 6 - 1 → 5 characters =LEFT(A2,5) → "Priya"

LEN — Count Characters

=LEN(text)
=LEN("Hello, world!") → 13 (counts spaces and punctuation too) =LEN(TRIM(A2)) → length after removing extra spaces

TRIM — Remove Extra Spaces (Most Common Cleaning Function)

Most Common Data Problem

Someone typed " Clothing " with spaces. TRIM removes all leading and trailing spaces and collapses internal double spaces to single.

=TRIM(" Clothing ") → "Clothing" =TRIM(" Credit Card") → "Credit Card"

UPPER, LOWER, PROPER — Fix Capitalization

=UPPER("kanyon") → "KANYON"

ALL CAPS

=LOWER("KANYON") → "kanyon"

all lowercase

=PROPER("john smith") → "John Smith"

First Letter Caps

CONCATENATE and the & Operator

=CONCATENATE(text1, text2)   OR   =A2 & " " & B2
🛍️ Build a label from data columns
If A2="Priya", B2="Sharma", C2="Clothing" =A2 & " " & B2 → "Priya Sharma" =A2 & " bought " & C2 → "Priya bought Clothing" =TEXTJOIN(", ",TRUE,A2:A5) → "Name1, Name2, Name3, Name4"

FIND — Locate Text Within Text

=FIND( find_this , look_inside , [start_position] )
=FIND("@","priya@email.com") → 6 (@ is 6th character) =FIND(" ","John Smith") → 5 (space at position 5)

Dates and Time in Excel

How Excel Stores Dates

Every date is stored as a serial number. January 1 1900 = 1. Today is a number like 45000+. This lets Excel subtract dates to find days between them. You see "05-Aug-2022" only because of number formatting.

TODAY and NOW
=TODAY() → today's date (auto-updates) =NOW() → current date + time
DAYS — count between dates
=DAYS(end_date, start_date) =DAYS(B2,A2) → days between A2 and B2 =TODAY()-A2 → days since a past date
DAY, MONTH, YEAR
=YEAR(A2) → 2022 =MONTH(A2) → 8 =DAY(A2) → 5
DATEDIF — Age Calculator
=DATEDIF(A2,TODAY(),"Y") → full years (perfect for age!)

Lookup and Conditional Functions

VLOOKUP and IF together automate what used to take hours. Master these and you will stand out immediately.

VLOOKUP — Search a Table Vertically

VLOOKUP looks for a value in the first column of a table and returns something from the same row — like searching a phone book by name to find the number.

=VLOOKUP( what_to_find , table_range , column_number , FALSE )
🛍️ Find category for a given invoice number
=VLOOKUP("I138884", $A$2:$I$201, 4, FALSE) ↑ ↑ ↑ ↑ Invoice ID Search here Col 4 Exact → Returns "Clothing" (4th column from A = column D = category)
ArgumentWhat to TypeExplanation
lookup_value"I138884" or a cell refWhat you are searching for
table_array$A$2:$I$201The full table — always lock with $
col_index_num4Count from the LEFT of the table range
range_lookupFALSEFALSE = exact match. Always use FALSE!
=VLOOKUP(A2,$A$2:$I$201,6,FALSE) ← get price for invoice in A2 =VLOOKUP(A2,$A$2:$I$201,9,FALSE) ← get shopping mall name
Three VLOOKUP Rules to Memorise

1. The search column MUST be the first column of your table_array. You can only look right, never left.

2. Always use FALSE (exact match) for the 4th argument — TRUE gives wrong results on unsorted data.

3. Lock your table range with $ (press F4) so it does not shift when you copy the formula down.

HLOOKUP — Horizontal Search

=HLOOKUP( what_to_find , table_range , row_number , FALSE )
If row 1 has: Jan, Feb, Mar, Apr If row 2 has: 1000, 1200, 900, 1500 =HLOOKUP("Mar", A1:D2, 2, FALSE) → 900 Searches across the TOP ROW, returns value from the specified row below.

IF — Make Decisions in Formulas

=IF( condition , value_if_TRUE , value_if_FALSE )
🎓 Student Pass or Fail
=IF(B2>=50, "Pass", "Fail") → Score 72 → "Pass" | Score 40 → "Fail" =IF(F2>1000, "Premium", "Standard") → Tag shopping transactions by value =IF(B2="Female", "F", "M") → Shorten gender labels

Nested IF — Multiple Conditions (Grade Calculator)

🎓 Grade Calculator from the Practice Sheet
=IF(H2>=90,"A++", IF(H2>=80,"A+", IF(H2>=70,"A", IF(H2>=60,"B+", IF(H2>=35,"B","F"))))) Score 95 → A++ | Score 82 → A+ Score 71 → A | Score 45 → B | Score 20 → F Tip: Count brackets! Each IF needs one opening ( and one closing ) There are 5 IFs above so you need 5 closing brackets at the end.

IFERROR — Handle Errors Gracefully

=IFERROR( formula_that_might_error , what_to_show_instead )
Why This Matters

VLOOKUP throws #N/A when it cannot find a value. IFERROR wraps it so users see "Not Found" instead of a scary error.

=IFERROR(VLOOKUP(A2,$A$2:$I$201,4,FALSE), "Not Found") → Returns category OR "Not Found" — never shows #N/A =IFERROR(B2/C2, 0) → Returns 0 instead of #DIV/0! when dividing by zero

Common Error Messages

ErrorMeaningCommon Cause
#N/AValue not foundVLOOKUP could not find the lookup value
#DIV/0!Division by zero=A1/B1 where B1 is empty or zero
#VALUE!Wrong data typeDoing math on text ("apple" + 5)
#REF!Invalid referenceDeleted a cell a formula was pointing to
#NAME?Unknown functionTypo in function name (=SUMM instead of =SUM)

Data Cleaning Techniques

80% of data work is cleaning, not analysing. Master these tools and save hours every week.

The 6 Most Common Data Problems

Problem 1 — Extra Spaces

" Clothing " looks same as "Clothing" visually but Excel treats them differently. VLOOKUP will fail to match.

Fix: =TRIM(A2)

Problem 2 — Inconsistent Case

"clothing", "CLOTHING", "Clothing" are all different to Excel. Filters miss rows.

Fix: =PROPER(A2) or =LOWER(A2)

Problem 3 — Duplicate Rows

Same invoice appearing twice inflates sums and counts.

Fix: Data tab → Remove Duplicates

Problem 4 — Blank Cells

Missing values cause #DIV/0! errors, wrong averages, broken formulas.

Fix: Ctrl+G → Special → Blanks

Problem 5 — Numbers Stored as Text

SUM returns 0, numbers left-aligned, green triangle in corner of cells.

Fix: =VALUE(A2) or Data → Text to Columns

Problem 6 — Text in One Column

"John Smith" when you need first and last name in separate columns.

Fix: Data → Text to Columns

Remove Duplicates

  1. Click any cell in the dataset
  2. Go to Data tab in the Ribbon
  3. Click Remove Duplicates
  4. Choose columns to check — tick "Customer Name" AND "Email" to find rows where both match
  5. Click OK — Excel tells you how many were removed
Important

Always make a backup copy on a separate sheet before removing duplicates. This cannot be easily undone once saved.

Find and Fill Blank Cells

  1. Select the entire column you want to check
  2. Press Ctrl+G → click Special → tick Blanks → OK
  3. All blank cells are now selected
  4. Type a value or formula, then press Ctrl+Enter to fill ALL blanks at once

Text to Columns — Split One Column Into Two

👤 Split "John Smith" into "John" and "Smith"
  1. Select the column with full names
  2. Data tab → Text to Columns
  3. Choose Delimited → Next
  4. Tick Space as the delimiter → Next
  5. Choose destination cell → Finish

Also works for "John,Smith" (use comma delimiter) or dates formatted as "2022-08-05" (use dash).

SUBSTITUTE — Find and Replace Inside Formulas

=SUBSTITUTE( text , old_text , new_text )
=SUBSTITUTE(A2," "," ") → replace double space with single =SUBSTITUTE(A2,"Dept.","Department") → standardize abbreviations =SUBSTITUTE(A2,"-","") → remove all hyphens

Complete Cleaning Workflow

Step-by-Step Checklist
  1. Remove duplicates (Data → Remove Duplicates)
  2. Trim spaces — create helper column with =TRIM(A2), copy → paste as values, delete original
  3. Fix capitalization with =PROPER() or =LOWER()
  4. Find blank cells with Ctrl+G → Blanks
  5. Convert text-numbers with =VALUE() or Text to Columns trick
  6. Verify: if =COUNT() and =COUNTA() differ for a column, you have text mixed with numbers
Pro Tip — Paste as Values

After cleaning with =TRIM or =PROPER, copy the cleaned column → right-click → Paste Special → Values Only. This replaces formulas with static clean text so you can delete the original messy column safely.

Data Visualization with Conditional Formatting

Make your data tell a story without building charts. Conditional formatting colors cells automatically based on their values.

The Core Idea

How It Works

You set a rule: "If price is greater than 1000, color it green." Excel applies the formatting automatically to every matching cell and updates it whenever data changes. No manual highlighting needed.

How to Apply Conditional Formatting

  1. Select the data range you want to format (e.g., F2:F201)
  2. Go to Home tab → Conditional Formatting
  3. Choose your rule type from the menu
  4. Set your condition and choose a color style
  5. Click OK

The 4 Types

1. Highlight Cell Rules

Color specific cells based on simple conditions.

RuleExample
Greater ThanPrice > 1000 → Green
Less ThanScore < 35 → Red
Equal ToStatus = "Pass" → Yellow
Text ContainsCategory has "Shoes" → Blue
2. Color Scales

Grades cells from one color to another based on value. Red to Yellow to Green (low to high).

Low value → High value

Perfect for: average scores, sales figures, age distribution.

3. Data Bars

Creates a mini bar chart inside each cell. Longer bar = higher value.

1800.51
600.16
60.60
4. Icon Sets

Adds arrows, traffic lights, or stars to categorize values.

▲ High (top 33%)

▶ Medium (middle 33%)

▼ Low (bottom 33%)

Traffic light icons give instant at-a-glance status.

Class Exercise — Student Result Sheet

🎓 Apply Yellow to Pass, Red to Fail in the Result column
  1. Select the Result column range (e.g., K2:K51)
  2. Home → Conditional Formatting → Highlight Cells Rules → Equal To
  3. Type "Pass" → Select "Yellow Fill with Dark Yellow Text" → OK
  4. Repeat: Equal To "Fail" → Select "Light Red Fill with Dark Red Text" → OK

Apply color scale to the Average column:

  1. Select the Average column range
  2. Conditional Formatting → Color Scales → More Rules
  3. Set Minimum color to Red and Maximum color to Green → OK
Managing Rules

To edit or delete rules: Home → Conditional Formatting → Manage Rules. You will see all rules on the selected range. Order matters — the first matching rule wins. Use the arrows to reorder them.

Sort, Filter, Subtotals and Tables

Organize, slice, and summarize your data without writing formulas. These are Excel most-used data management tools.

Sorting Data

Quick Sort — one column
  1. Click any cell in the column to sort by
  2. Data tab → Sort A to Z or Sort Z to A
  3. The entire row moves together — all columns stay aligned
Multi-Level Sort
  1. Data tab → Sort (the full dialog)
  2. Sort by "Region" A to Z
  3. Click Add Level → sort by "Sales" largest to smallest
  4. OK — sorts by region first, then by sales within each region
🛍️ Sort shopping data by age (youngest to oldest)
  1. Click any cell in the dataset
  2. Data → Sort → Sort by "age" → Cell Values → Smallest to Largest → OK

Invoice ID, gender, category all stay aligned with the correct age automatically.

Filtering — Show Only What You Need

  1. Click any cell in the header row
  2. Data tab → Filter (dropdown arrows appear on every header)
  3. Click any dropdown → untick values you do not want → OK
  4. Filtered rows are hidden (still exist, just not shown)
  5. To remove: Data → Clear, or click the filter icon again
Filter by Text
Gender dropdown: ☑ Male ☐ Female ← untick Only Male rows visible. Other rows still exist, just hidden.
Filter by Number
Price column → Number Filters → Greater Than → type 1000 Shows only purchases over 1000 Or use "Top 10" to see the highest-priced items.

Subtotals — Automatic Group Summaries

What Subtotals Do

Subtotals insert summary rows automatically after each group change. For example: a "Total Sales" row after every region group — without you writing a single formula.

  1. Sort first by the column you want to group by (e.g., Region)
  2. Data tab → Subtotal
  3. "At each change in" → select "Region"
  4. "Use function" → Sum
  5. "Add subtotal to" → tick "Sales" → OK

Click the 1, 2, 3 outline buttons on the left to collapse and expand groups.

Converting Data to a Table

Tables are Smarter than Plain Ranges

Benefits of Tables

  • Auto-expand when you add rows
  • Filter dropdowns built-in
  • Formulas auto-copy to new rows
  • Structured references: =SUM(Table1[Price])

When to use plain ranges

  • Small static lookup tables
  • Reference data that will not grow
  • Complex print layouts

Create a Table

  1. Click any cell in your data
  2. Insert tab → Table, or press Ctrl+T
  3. Check "My table has headers" → OK
  4. Rename it: Table Design tab → type a name like "SalesData"

SUBTOTAL Function vs Regular SUM

Use SUBTOTAL when your data is filtered

=SUM(F2:F201) still adds ALL rows including hidden ones after filtering. Use =SUBTOTAL to only sum visible rows.

=SUBTOTAL(9, F2:F201) ← 9 = SUM of visible rows only =SUBTOTAL(1, F2:F201) ← 1 = AVERAGE of visible rows =SUBTOTAL(2, F2:F201) ← 2 = COUNT of visible rows

Add-on Section — Beyond the Basics

These topics complement everything you have learned today and come up constantly in real jobs.

1. Freeze Panes — Keep Headers Visible While Scrolling

Must-Know Practical Skill

When you scroll down 100 rows, column headers disappear. Freeze them so they are always visible.

  1. Click cell A2 (one row below the header row)
  2. View tab → Freeze Panes → Freeze Panes
  3. Scroll down — row 1 stays frozen at the top!

To freeze columns too: click cell B2 before freezing. To unfreeze: View → Freeze Panes → Unfreeze Panes.

2. Named Ranges — Give Cells Friendly Names

Write Readable Formulas

Instead of =F2:F201, name it "Prices". Your formulas become readable English.

Create: Select F2:F201 → click Name Box (top left) → type "Prices" → Enter Now use it anywhere: =SUM(Prices) ← instead of =SUM(F2:F201) =SUMIF(Category,"Shoes",Prices) ← much clearer! Manage all names: Formulas tab → Name Manager

3. Data Validation — Drop-Down Lists

Prevent Wrong Data Entry

Create a drop-down so users can only pick from valid options. No typos, no inconsistent values.

  1. Select the column where data will be entered
  2. Data tab → Data Validation
  3. Allow: List → Source: type "Clothing,Shoes,Books,Cosmetics"
  4. OK — a dropdown arrow now appears in every cell in that range

Or point to a range: Source = $D$1:$D$6 to use a list stored in another column.

4. Introduction to Pivot Tables

The Most Powerful Excel Feature

Pivot Tables summarize thousands of rows by dragging and dropping — no formulas needed at all.

🛍️ How much did each category make? Split by gender?
  1. Click any cell in the shopping data
  2. Insert tab → Pivot Table → OK (new sheet is created)
  3. Drag "category" to the Rows area
  4. Drag "price" to the Values area (auto-sums!)
  5. Drag "gender" to Columns for a breakdown by gender

5. Flash Fill — Excel Learns Your Pattern

AI-Powered Extraction

Type one example manually. Press Ctrl+E. Excel recognizes the pattern and fills the rest of the column.

Column A: "I138884" (invoice IDs) You want: "138884" (number only) Step 1: In B2, manually type: 138884 Step 2: Press Ctrl + E Excel fills B3, B4, B5... automatically! Works for: names, phone numbers, dates, codes — any consistent pattern.

6. XLOOKUP — The Modern VLOOKUP (Office 365)

No More VLOOKUP Limitations

XLOOKUP fixes every VLOOKUP problem: can look left, handles missing values built-in, simpler syntax.

=XLOOKUP( what_to_find , search_column , return_column , [if_not_found] )
=XLOOKUP(A2, $A$2:$A$201, $F$2:$F$201, "Not Found") ↑ ↑ ↑ ↑ Search for Search here Return this If missing No column numbers to count! Can search left, right, or in any direction. Error handling built-in — no IFERROR wrapper needed.
